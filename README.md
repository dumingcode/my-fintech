本项目为投资平台服务端。
## 功能目标
1. 实现从第三方网站通过API的方式获取指数估值数据。
2. 按照盈利收益率法、博格公式衡量指数估值。
3. 根据指数估值确定每月定投额度，便宜的时候多买，贵的时候少买。

## 技术方案
- backend:koa2
- frontend:vuejs
- datacenter:nodejs
